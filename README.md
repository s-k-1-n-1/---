# Weather & System Monitor

Этот проект представляет собой веб-приложение для отслеживания погоды и получения системных данных. Система состоит из интерактивного фронтенда и серверного бэкенда на Node.js.

---

##  Техническое задание (ТЗ)

### 1. Цель проекта
Создание программного комплекса для визуализации внешних данных (погода) и внутренней системной информации через собственный прокси-сервер.

### 2. Стек технологий
- **Frontend:** HTML5, CSS3, JavaScript (Fetch API).
- **Backend:** Node.js, Express.js.
- **Интеграция:** REST API, протокол HTTP (порт 3000).

### 3. Функциональные возможности
- **Модуль Погоды:** Получение данных напрямую из OpenWeatherMap API.
- **Модуль Интеграции:** Получение статуса и системных сообщений с локального Node.js сервера.
- **Динамика:** Автоматическое обновление интерфейса при получении данных с бэкенда.
